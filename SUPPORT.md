# Support

Thanks for using **Hex Viewer**! Here is how to get help.

## Documentation

- [README](./README.md) — features, installation, and usage.
- [CONTRIBUTING](./CONTRIBUTING.md) — development setup and scripts.
- [AI-generated docs (DeepWiki)](https://deepwiki.com/DoctorLai/hex-viewer).

## Questions & Discussions

- Open a [GitHub Discussion](https://github.com/DoctorLai/hex-viewer/discussions)
  for questions, ideas, and general help. (If Discussions are not enabled, use an
  issue instead.)

## Bugs & Feature Requests

- Found a bug? Open a [bug report](https://github.com/DoctorLai/hex-viewer/issues/new?template=bug_report.md).
- Have an idea? Open a [feature request](https://github.com/DoctorLai/hex-viewer/issues/new?template=feature_request.md).

Before filing, please search [existing issues](https://github.com/DoctorLai/hex-viewer/issues)
to avoid duplicates.

## Security
